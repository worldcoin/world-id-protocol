# World ID

World ID is an anonymous proof of human for the age of AI.

This crate serves as the main entry point for the World ID Rust SDK. Currently, it primarily re-exports functionality from the `world-id-core` crate.

More information can be found in the [World ID Developer Documentation](https://docs.world.org/world-id).

## Installation

Add this to your `Cargo.toml`:

```toml
[dependencies]
world-id = "0.1.0"
```
