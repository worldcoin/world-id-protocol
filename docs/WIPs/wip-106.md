## Abstract

This spec introduces the concept of Authenticator Attestations which are authenticated statements from World ID [Authenticators](https://docs.rs/world-id-authenticator/0.9.0/world_id_authenticator/struct.Authenticator.html) where the authenticator vendor provides optional information about the code integrity of the authenticator, the hardware integrity of where the authenticator is running, additional risk information, and information on holder authentication mechanisms. These statements may accompany World ID Proofs.


## Motivation

Some use cases require additional information beyond the World ID Proof that provides information on the environment in which the proof is generated. This is useful for use cases with high assurance requirements. One particular use case of this is requiring FaceAuth checks for proof presentation. FaceAuth compares locally the holder of a World ID Credential with the human in front of the device. Given these checks are done locally for privacy reasons, information about the integrity of the environment is required to provide assurance on the process.
